Задан массив чисел X(n). Реализуйте в программе:
1. Элементы массива целые числа. Размер массива 10. Ввод элементов массива. Вывод на экран сумму элементов массива
2. Элементы массива целые числа. Ввод с клавиатуры размера массива. Ввод элементов массива. Вывод на экран отрицательных элементов массива
3. Элементы массива вещественные числа. Размер массива 10. Ввод элементов массива. Вывод на экран произведения элементов массива
4. Элементы массива целые числа. Ввод с клавиатуры размера массива. Ввод элементов массива.  Вывод на экран суммы отрицательных элементов массива.
